# PicShare

PicShare is a igmur's based social network.

## Installation

Clone this repository:
```
git clone https://github.com/alejandro0619/PicShare.git
```
Use the package manager [npm](https://www.npmjs.com) to install dependencies.

```bash:
 npm install
```

## Usage
To run this project:
```bash
npm start
```
